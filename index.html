> The Filter & Policies

!

The Anti-Fragility Clause

Do not join if you have a fragile ego or expect to be babied. You will receive direct, unfiltered code reviews. If your code is garbage, we will tell you it is garbage, why it is garbage, and how to fix it.

Example Feedback: "This implementation is inefficient and will bottleneck at scale. Tear down lines 40-85 and rewrite using [Design Pattern]. Read the docs."

Public Accountability: All results, PRs, and critical feedback are posted publicly within the cohort. If you cannot handle public critique, do not apply.
